# 🛠️ Tech Skills for LLM — Beginner to Expert

**Self-contained interactive course:** Master every technical skill needed to understand and build LLMs. No external searching required (except Python installation).

## 3 Progressive Levels, 12 Topics

### 🔵 Level 1: Foundations (Beginner)
| # | Topic | What You'll Learn | LLM Connection |
|---|-------|-------------------|----------------|
| 1 | **Vectors** | Lists of numbers, dot product, cosine similarity | Every word = a vector |
| 2 | **Matrices** | Matrix multiply, shapes, transpose, broadcasting | Every layer = matrix multiply |
| 3 | **Derivatives** | Power rule, slopes, gradient descent | How models learn (minimize loss) |
| 4 | **Probability** | Softmax, temperature, cross-entropy, sampling | How models predict next tokens |

### 🟣 Level 2: Building Blocks (Intermediate)
| # | Topic | What You'll Learn | LLM Connection |
|---|-------|-------------------|----------------|
| 5 | **Backpropagation** | Chain rule, gradient flow, vanishing gradients | How millions of weights get updated |
| 6 | **Neurons & Layers** | Activations (ReLU, GELU, SwiGLU), loss functions | Building blocks of every network |
| 7 | **NLP Pipeline** | Tokenization (char, word, BPE), embeddings, positions | Text → numbers pipeline |
| 8 | **PyTorch** | Tensors, autograd, nn.Module, training loop | The framework to build it all |

### 🟠 Level 3: LLM-Ready (Advanced)
| # | Topic | What You'll Learn | LLM Connection |
|---|-------|-------------------|----------------|
| 9 | **Optimizers** | SGD → Adam → AdamW, LR schedules, gradient clipping | The training recipe |
| 10 | **Attention** | Q/K/V, scaled dot-product, multi-head, causal mask | THE innovation behind Transformers |
| 11 | **Transformers** | Full architecture, LayerNorm, residuals, FFN | The complete model stack |
| 12 | **Training** | Data prep, batching, evaluation, generation | End-to-end pipeline |

## How to Use

Open `tech_skills_interactive.html` in any modern browser. No installation needed!

- **Left panel:** Complete Python code with detailed comments
- **Right panel:** Explanations, interactive games, quizzes, and visualizations
- **Score tracking:** Earn points across all 12 topics
- **Level progression:** L1 → L2 → L3 builds on previous knowledge

## Course Design

- **Self-contained:** Every concept explained from first principles with analogies
- **Interactive:** Canvas visualizations, step-by-step calculators, drag-and-click games
- **Progressive:** Each level assumes knowledge from the previous level
- **LLM-focused:** Every topic connects directly to how LLMs work
